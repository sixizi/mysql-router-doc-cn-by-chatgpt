MySQL Router 是构建高可用性（HA）解决方案的构件。它通过智能路由连接到 MySQL 服务器，简化了应用程序开发，提高了性能和可靠性。

MySQL Router 8 全面支持 MySQL 5.7 和 MySQL 8，并且取代了 MySQL Router 2.x 系列。如果您当前使用 Router 2.0 或 2.1，我们建议您将安装升级到 MySQL Router 8。
